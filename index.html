Tutorial Git

Principais comandos Git

Git Clone

Comando utilizado para baixar o repositório para sua máquina e poder versionar lá pelo git

Git Checkout

Comando utilizado para posicionar no branh que deseja para fazer suas devidas atualizações.

Git Commit

Comando utilizado para para salvar suas devidas atualizações.

Git push

Comando utilizado para enviar as alterações para o repositório remoto.
